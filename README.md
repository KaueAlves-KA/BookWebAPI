# Book Review API

A Web API for books. You'll handle a book database using all the standard CRUD operations that a complete Web API needs.
(C# .NET WEB API)

## Technologies used
C#, .NET 8, .NET Framework 4.8, Entity Framework Core, SQL Server, AutoMapper. 

## Prerequisites
SQL Server EXPRESS / SQL Server localDB

## API Documentation (Swagger/OpenAPI)
This API is documented using **Swagger/OpenAPI**. The interactive documentation allows you to test the endpoints directly from your browser.

## Important to know
Place your connection string in the appsettings.json file where indicated.
The migration files for creating the database with Entity Framework are already included.
Remember to run the command to create the database and apply the Seed.cs file.
